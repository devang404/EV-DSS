## üìñ Introduction

The **Electric Vehicle Decision Support System (EV-DSS)** is a state-of-the-art web application developed to empower consumers, policymakers, and businesses with accurate, data-backed insights into Electric Vehicle adoption. By analyzing Total Cost of Ownership (TCO), environmental impact, and infrastructure readiness, EV-DSS clarifies the economic and ecological benefits of switching to EVs in the Indian context.

## ‚ú® Key Features

*   **üí∞ TCO Calculator**: Compare the lifetime costs of EV vs. ICE vehicles, factoring in fuel, maintenance, and subsidies.
*   **üåç Environmental Impact Analysis**: Visualize CO‚ÇÇ emission reductions and environmental benefits.
*   **ü§ñ AI Scenario Modeling**: Interactive "What-If" scenarios to explore the impact of changing petrol prices, electricity rates, and subsidies.
*   **üìä Interactive Dashboard**: A user-friendly interface visualizing complex data for easy decision-making.
*   **üó∫Ô∏è City Clustering**: ML-powered readiness assessment for different Indian cities.
*   **üß† AI Assistant**: A built-in chatbot to answer user queries regarding EV transition and policy.

## üõ†Ô∏è Installation & Setup

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/devang404/EV-DSS.git](https://github.com/devang404/EV-DSS.git)
    cd EV-DSS
    ```

2.  **Install Dependencies**
    ```bash
    npm install
    ```

3.  **Environment Setup**
    Create a `.env` file in the root directory and add your Supabase credentials:
    ```env
    VITE_SUPABASE_URL=your_supabase_url
    VITE_SUPABASE_PUBLISHABLE_KEY=your_supabase_key
    ```

4.  **Run Development Server**
    ```bash
    npm run dev
    ```
